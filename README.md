# Nubble App

This is an Social Network project built with **TypeScript**, **React Native**, **React Native CLI**, and **@shopify/restyle**. It aims to replicate the core functionality and design of the popular social media app like Instagram.

## Features

- User authentication: Users can sign up, log in, and log out.
- Profile customization: Users can update their profile information, including their profile picture and bio.
- News feed: Users can view a scrollable feed of posts from the people they follow.
- Post creation: Users can create new posts by uploading photos and adding captions.
- Like and comment: Users can interact with posts by liking and commenting on them.
- Follow and unfollow: Users can follow and unfollow other users to see their posts in their news feed.

## Installation

1. Clone the repository: <br />
   `git clone https://github.com/GersonRocha9/NubbleApp`

2. Navigate to the project directory: <br />
   `cd NubbleApp`

3. Install the dependencies: <br />
   `yarn`

4. Configure the development environment by following the provided guidelines and setting up necessary credentials/API keys.

5. Start the development server: <br />
   `yarn start`

6. Run the app on a connected device or emulator: <br />
   `yarn android` # for Android <br />
   `yarn ios` # for iOS

## Technologies Used

- **TypeScript**: a statically typed superset of JavaScript that helps improve code quality and developer productivity.
- **React Native**: a cross-platform framework for building mobile applications using React and native components.
- **React Native CLI**: a command-line interface for creating and managing React Native projects.
- **@shopify/restyle**: a utility library for building consistent and themeable UI components in React Native.
